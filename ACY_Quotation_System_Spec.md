# 12. ระบบต้นทุนและกำไร

ระบบภายในต้องคำนวณ:

```text
ต้นทุนสินค้า
ต้นทุนติดตั้ง
ต้นทุนบริการ
ต้นทุนขนส่ง
ต้นทุนรวม
ราคาขาย
ส่วนลด
ราคาขายสุทธิ
กำไร
GP%
Markup%
```

สูตร:

```text
กำไร = ราคาขายสุทธิ - ต้นทุนรวม

GP% = (กำไร / ราคาขายสุทธิ) × 100

Markup% = (กำไร / ต้นทุนรวม) × 100
```

## ข้อกำหนด

- ราคาทุนห้ามแสดงใน PDF ลูกค้า
- กำไรห้ามแสดงใน PDF ลูกค้า
- เฉพาะผู้มีสิทธิ์จึงจะดูต้นทุนได้
- ตั้งค่า Minimum GP ได้
- หาก GP ต่ำกว่าเกณฑ์ ต้องขออนุมัติ
- ส่วนลดเกินกำหนดต้องขออนุมัติ
- บันทึกชื่อผู้อนุมัติ วันที่ และเหตุผล
