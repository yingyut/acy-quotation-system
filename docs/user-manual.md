## 1. เข้าสู่ระบบและเปลี่ยนรหัสผ่าน

เปิดเบราว์เซอร์ไปที่ URL ที่ผู้ดูแลระบบแจ้ง (เช่น `http://192.168.1.50:8090`) กรอกชื่อผู้ใช้และรหัสผ่าน

หากเข้าสู่ระบบครั้งแรกด้วยรหัสผ่านเริ่มต้น ระบบจะแจ้งเตือนให้เปลี่ยนรหัสผ่าน — คลิกชื่อของท่านที่มุมขวาบน → **เปลี่ยนรหัสผ่านของฉัน**

ระบบจะออกจากระบบอัตโนมัติหากไม่มีการใช้งานเกิน 8 ชั่วโมง (ปรับได้โดยผู้ดูแลระบบ)

## 2. ภาพรวม Dashboard

หน้าแรกหลัง Login แสดงสรุปข้อมูลสำคัญ: จำนวน/มูลค่าใบเสนอราคาวันนี้และเดือนนี้, อัตราปิดการขาย, เอกสารรออนุมัติ, ใบแจ้งหนี้ค้างชำระเกินกำหนด และ (สำหรับผู้มีสิทธิ์ดูต้นทุน) กำไรและ GP เฉลี่ย

## 3. จัดการข้อมูลลูกค้า

เมนู **ลูกค้า**:

- **เพิ่มลูกค้า** — กรอกข้อมูล รหัส, ชื่อ, ที่อยู่, เลขผู้เสียภาษี ฯลฯ ระบบจะแจ้งเตือนหากเลขผู้เสียภาษีซ้ำกับลูกค้าที่มีอยู่แล้ว
- **ผู้ติดต่อ** — เพิ่มผู้ติดต่อได้หลายคนต่อลูกค้าหนึ่งราย (จากหน้ารายละเอียดลูกค้า)
- **ค้นหา** — ค้นหาจากชื่อ, รหัส, เลขผู้เสียภาษี, เบอร์โทร
- **Import/Export Excel** — กด "ดาวน์โหลด Template" เพื่อได้ไฟล์ตัวอย่าง กรอกข้อมูลแล้วอัปโหลดที่หน้า Import ระบบจะตรวจสอบและแสดง Preview ก่อนนำเข้าจริงเสมอ แถวที่มีปัญหาจะไม่ถูกนำเข้าและแสดงสาเหตุให้แก้ไข
- **ประวัติเอกสาร** — หน้ารายละเอียดลูกค้าแสดงใบเสนอราคาทั้งหมดของลูกค้ารายนั้น

## 4. จัดการข้อมูลสินค้าและบริการ

เมนู **สินค้าและบริการ**:

- กรอกข้อมูลทั่วไป, ราคาขาย (ราคาสินค้า/ติดตั้ง/มาตรฐาน) และ **ราคาทุน** (แสดงเฉพาะผู้มีสิทธิ์ — Admin, Sales Manager, หรือ Sales ที่ได้รับสิทธิ์พิเศษ)
- **รูปสินค้า** — อัปโหลดได้หลายรูป ตั้งรูปหลักได้
- **Bullet Specification** — เพิ่มรายการ Spec แบบหัวข้อ-ค่า (เช่น "CPU: Intel i5")
- **Datasheet** — แนบไฟล์ PDF ได้
- **Duplicate** — คัดลอกสินค้าเดิมเพื่อสร้างรายการใหม่ที่คล้ายกันได้รวดเร็ว
- **ประวัติการเปลี่ยนราคา** — บันทึกทุกครั้งที่แก้ไขราคาทุน/ราคาขาย (มองเห็นเฉพาะผู้มีสิทธิ์ดูต้นทุน)

## 5. สร้างใบเสนอราคา

1. เมนู **ใบเสนอราคา** → **+ สร้างใบเสนอราคา** → ค้นหาและเลือกลูกค้า ระบบจะสร้างเอกสารพร้อมเลขที่อัตโนมัติทันที และเปิดหน้าแก้ไข
2. กรอกข้อมูลหัวเอกสาร: ผู้ติดต่อ, ชื่อโครงการ, หัวข้อ, วันที่, จำนวนวันยืนราคา, เครดิตเทอม, เงื่อนไขการชำระเงิน, Template
3. เพิ่มรายการโดยเลือกประเภทที่ต้องการ:
   - **+ สินค้า / + บริการ** — ค้นหาจากฐานข้อมูลสินค้า (ราคา/ต้นทุนจะถูกดึงมาให้อัตโนมัติ) หรือพิมพ์ชื่อรายการเองก็ได้
   - **+ ข้อความ** — สำหรับรายละเอียดที่ไม่คิดราคา เช่น "ค่าเดินทาง รวมในราคาแล้ว"
   - **+ หัวข้อหมวด** — สำหรับแบ่งกลุ่มรายการ เช่น "งานติดตั้ง", "งานเดินสาย"
   - **+ เหมารวม** — ใส่ราคารวมเดียว ซ่อนราคาต่อหน่วยจากลูกค้า (แสดงเป็น "-" ในเอกสาร)
4. ลากรายการ (จับที่ไอคอน ⠿) เพื่อจัดลำดับใหม่ได้ตามต้องการ กด "Duplicate" เพื่อคัดลอกรายการ
5. ระบบคำนวณยอดรวม/ส่วนลด/VAT/หัก ณ ที่จ่าย/ยอดสุทธิแบบ Real-time ที่ด้านล่างของหน้า พร้อม (สำหรับผู้มีสิทธิ์) ต้นทุนรวม กำไร GP% และ Markup%
6. กด **"บันทึกร่าง (Draft)"** เพื่อบันทึกไว้ทำต่อภายหลัง หรือ **"บันทึกและส่งอนุมัติ"** เมื่อพร้อม

## 6. ขออนุมัติและอนุมัติใบเสนอราคา

ระบบจะกำหนดให้ต้องขออนุมัติจาก Sales Manager โดยอัตโนมัติเมื่อ:
- GP% ต่ำกว่าเกณฑ์ที่กำหนด (ค่าเริ่มต้น 15% ปรับได้โดย Admin), หรือ
- ส่วนลด (รายการหรือท้ายบิล) เกินเกณฑ์ที่อนุมัติเองได้ (ค่าเริ่มต้น 15%)

หากไม่เข้าเงื่อนไขข้างต้น เอกสารจะอนุมัติอัตโนมัติทันทีที่กด "ส่งอนุมัติ"

**สำหรับ Sales Manager:** เมนู "ใบเสนอราคา" จะเห็นเอกสารสถานะ "รออนุมัติ" ทั้งหมด เปิดเอกสารแล้วกด **"อนุมัติ"** หรือ **"ไม่อนุมัติ"** (ต้องระบุเหตุผล — เอกสารจะกลับไปเป็นสถานะร่างให้ Sales แก้ไข)

## 7. Preview / Export / พิมพ์ PDF

จากหน้ารายละเอียดใบเสนอราคา:

- **Preview PDF** — เปิดดูตัวอย่างในเบราว์เซอร์ (ไม่บันทึกลง Audit Log)
- **Export PDF** — ดาวน์โหลด/เปิดไฟล์ PDF ต้นฉบับ
- **พิมพ์ต้นฉบับ+สำเนา** — รวมต้นฉบับและสำเนา (มีข้อความ "ต้นฉบับ/ORIGINAL" หรือ "สำเนา/COPY" กำกับ) ไว้ในไฟล์ PDF เดียวกัน พร้อมพิมพ์ทันที

เอกสารที่ยาวเกิน 1 หน้าจะแบ่งหน้าอัตโนมัติ พร้อมแสดง Logo, ชื่อบริษัท, เลขที่เอกสาร, ชื่อลูกค้า และหัวตารางซ้ำทุกหน้า ยอดรวมและลายเซ็นจะแสดงเฉพาะหน้าสุดท้ายเสมอ

## 8. ส่งใบเสนอราคาให้ลูกค้าและปิดงาน

หลังอนุมัติแล้ว หน้ารายละเอียดจะมีปุ่มดำเนินการตามสถานะปัจจุบัน:

- **ส่ง Email ลูกค้า** — เปิดโปรแกรมอีเมลของท่านพร้อมแนบลิงก์ดาวน์โหลด PDF ให้อัตโนมัติ (ระบบไม่ได้ส่งอีเมลเอง ท่านตรวจสอบและกดส่งจากอีเมลของท่านเอง) และปรับสถานะเอกสารเป็น "ส่งลูกค้าแล้ว"
- **ลูกค้าตอบรับ** — เมื่อลูกค้าตกลง
- **ปิดงาน (ชนะ)** / **ปิดงาน (แพ้)** — บันทึกผลสุดท้าย กรณีแพ้งานต้องระบุเหตุผล
- **สร้าง Revision ใหม่** — เมื่อลูกค้าขอแก้ไขราคา/รายการ ระบบจะสร้างเอกสารเวอร์ชันใหม่ (Rev.1, Rev.2, ...) โดยยังใช้เลขที่เอกสารเดิม เวอร์ชันเก่าจะถูกเก็บไว้ดูย้อนหลังได้เสมอ ไม่มีการเขียนทับ
- **Duplicate** — คัดลอกทั้งฉบับเป็นใบเสนอราคาใหม่ (เลขที่ใหม่) สำหรับงานที่คล้ายกัน
- **ยกเลิกเอกสาร** — เลขที่เอกสารจะไม่ถูกนำกลับมาใช้ซ้ำแม้ยกเลิกแล้ว

## 9. แปลงเป็นใบสั่งขาย → ใบส่งสินค้า → ใบแจ้งหนี้

เมื่อใบเสนอราคาอยู่ในสถานะอนุมัติแล้วขึ้นไป (อนุมัติแล้ว/ส่งลูกค้าแล้ว/ลูกค้าตอบรับ/ชนะงาน) จะมีปุ่ม:

- **แปลงเป็นใบสั่งขาย** — คัดลอกรายการทั้งหมดมาสร้างใบสั่งขาย ไม่ต้องกรอกซ้ำ
- จากหน้าใบสั่งขาย: **สร้างใบส่งสินค้า** เมื่อจัดส่งสินค้า และ **สร้างใบแจ้งหนี้** เมื่อพร้อมเรียกเก็บเงิน
- หรือกด **แปลงเป็นใบแจ้งหนี้** จากหน้าใบเสนอราคาโดยตรงก็ได้ (ข้ามขั้นตอนใบสั่งขาย สำหรับงานที่ไม่ต้องการติดตามการจัดส่งแยก)

## 10. บันทึกรับชำระเงินและออกใบเสร็จ

จากหน้ารายละเอียดใบแจ้งหนี้ (สิทธิ์ Accounting/Admin):

1. กรอกแบบฟอร์ม "บันทึกการรับชำระเงิน": วันที่, จำนวนเงิน (ชำระได้บางส่วนหรือครบ), วิธีชำระ, เลขอ้างอิง
2. ระบบจะ**ออกใบเสร็จรับเงินให้อัตโนมัติ**ทันทีที่บันทึกการชำระเงินสำเร็จ พร้อมเลขที่เอกสารของตัวเอง
3. สถานะใบแจ้งหนี้จะเปลี่ยนเป็น "ชำระบางส่วน" หรือ "ชำระครบแล้ว" ตามยอดที่บันทึก ยอดคงเหลือคำนวณอัตโนมัติ
4. เปิดใบเสร็จจากเมนู **ใบเสร็จรับเงิน** เพื่อ Preview/Export/พิมพ์ต้นฉบับ+สำเนาได้เช่นเดียวกับใบเสนอราคา
5. หากลูกค้าต้องการใบกำกับภาษีแยก กด **"ออกใบกำกับภาษี"** จากหน้าใบแจ้งหนี้

## 11. รายงานและ Audit Log

- เมนู **รายงาน** — ยอดขายแยกตามพนักงาน/ลูกค้า/เดือน และสินค้าขายดี (คำนวณจากใบเสนอราคาที่ปิดงานสำเร็จ)
- เมนู **Audit Log** (Admin เท่านั้น) — ประวัติการสร้าง/แก้ไข/ลบ/Export/เปลี่ยนสถานะ/รับชำระเงินทุกรายการ พร้อมชื่อผู้ทำรายการและเวลา กรองตามประเภทข้อมูลหรือการกระทำได้ ข้อมูลนี้แก้ไขหรือลบไม่ได้โดยผู้ใช้ทั่วไป

## 12. การตั้งค่าสำหรับผู้ดูแลระบบ

เมนู **ผู้ดูแลระบบ** (มองเห็นเฉพาะ Admin):

- **ผู้ใช้งานและสิทธิ์** — เพิ่ม/ปิดใช้งานผู้ใช้ กำหนดบทบาท และอนุญาตให้ Sales รายบุคคลดูราคาทุนได้เป็นกรณีพิเศษ
- **ตั้งค่าบริษัท** — โลโก้, ชื่อบริษัท, ที่อยู่, เลขผู้เสียภาษี, ลายเซ็น, ตราประทับ, บัญชีธนาคาร, สาขา, เงื่อนไขมาตรฐาน, Footer, สีเอกสาร
- **ตั้งค่าเลขที่เอกสาร** — กำหนด Prefix, รูปแบบปี (พ.ศ./ค.ศ./ไม่แสดง), เดือน, วัน, จำนวนหลัก, และรอบการเริ่มเลขใหม่ แยกตามประเภทเอกสารแต่ละชนิด
- **ตั้งค่า Template** — ปรับ Logo, สี, ขนาดตัวอักษร, ระยะขอบ, การแสดงรูปสินค้า/รหัสสินค้า/ราคา/ส่วนลด/เลขหน้า ของ Template เอกสารแต่ละแบบ (ไม่ต้องแก้ไข Code หรือ HTML)
- **Backup / Restore** — ดู [คู่มือ Backup](backup-guide.md) และ [คู่มือ Restore](restore-guide.md)
- **Audit Log** — ตามหัวข้อ 11

## 13. คำถามที่พบบ่อย

**ถาม: ทำไม Sales ถึงมองไม่เห็นราคาทุนและ GP?**
ตอบ: เป็นการออกแบบตามสิทธิ์ (RBAC) เพื่อป้องกันข้อมูลอ่อนไหว บทบาท Sales จะเห็นเฉพาะราคาขาย Admin สามารถให้สิทธิ์พิเศษเป็นรายบุคคลได้ที่หน้า "ผู้ใช้งานและสิทธิ์" (ติ๊ก "อนุญาตให้ดูราคาทุน")

**ถาม: แก้ไขใบเสนอราคาที่อนุมัติแล้วได้ไหม?**
ตอบ: ไม่ได้โดยตรง เพื่อรักษาความถูกต้องของเอกสารที่อนุมัติแล้ว ให้กด "สร้าง Revision ใหม่" แทน ซึ่งจะสร้างเวอร์ชันใหม่ที่แก้ไขได้ โดยยังเก็บเวอร์ชันเดิมไว้อ้างอิง

**ถาม: เลขที่เอกสารจะซ้ำกันไหมถ้ามีคนกดสร้างพร้อมกันหลายคน?**
